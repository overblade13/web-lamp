# Web-Lamp

Интеллектуальная система управления освещением на базе ESP32, Node.js и Supabase.

## Особенности
- Удаленное управление яркостью и питанием через веб-интерфейс.
- Автоматический режим (по датчику освещенности).
- Синхронизация состояния в реальном времени.
- Современный дизайн (LUMO ONE).

## Структура проекта
- `/api` - Backend (Express.js)
- `/public` - Frontend (HTML/CSS/JS)
- `/src` - Прошивка ESP32 (PlatformIO)
- `wokwi.toml` - Конфигурация для симуляции в Wokwi
